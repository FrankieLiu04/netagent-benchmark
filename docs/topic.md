# YBL1: Vibe Networking — Network Engineering

- **Context**: Research prototype for LLM-driven network engineering
- **Area**: LLM agents for network engineering

## Description

This project explores the potential and capabilities of applying LLM to layer 2 (switching) and layer 3 (routing) network engineering. We adopt Cisco's Modelling Lab (CML) network emulator which has an open-source MCP server that can interface with LLM platforms.

## Three Core Goals

1. **LLM Benchmark for Network Configuration** — Assess the performance of existing LLMs on network configuration tasks by developing an LLM benchmark specifically for this use case.
2. **Identify Limitations & Develop Solutions** — Find LLM shortcomings in network configuration and create targeted improvements.
3. **LLM-Driven Automated Network Troubleshooting** — Explore autonomous network fault diagnosis and repair using LLMs.
